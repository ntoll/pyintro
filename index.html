Lists (of simple things)


    >>> shopping_list = ['ham', 'eggs', 'fruit', 'milk']
    >>> shopping_list
    ['ham', 'eggs', 'fruit', 'milk']

Lists remain in order.


    >>> shopping_list[0]
    'ham'
    >>> shopping_list[3]
    'milk'
    >>> shopping_list[4]
    Traceback (most recent call last):
    File "", line 1, in 
    IndexError: list index out of range

Refer to an item by its index. Python counts from 0 (zero).

If you index the wrong thing you get an error.

Dictionaries of keys & values


    >>> me = {
    ...   'name': 'Nicholas Tollervey',
    ...   'age': 39,
    ...   'languages': ['Python', 'Javascript', 'C#', 'Visual Basic']

    >>> me['languages'][0]

Open an Editor

Now type the following:


    #!/usr/bin/env python
    """
    A simple program to demonstrate flow.
    """

    while True:
        name = raw_input('What is your name? ')
        if len(name) > 0:
            print('Hello, %s' % name)
            break
        else:
            print('Please try again.')

Loops


    >>> shopping = ['ham', 'eggs', 'milk', 'sugar', 'spam']
    >>> for item in shopping:
    ...     print(item)
    ...
    ham
    eggs
    milk
    sugar
    spam

How would you describe this code..?

For each item in the shopping list, print it!

Python loops over each thing in "shopping", calls it "item" then does something with that item (print it).

Functions

Functions are discreet blocks of code that do something (and give you back a result). Often you pass input "arguments" in to a function.


    def hello(name='World'):
        """
        This function takes a name argument (that defaults to 'World')
        and returns a greeting.
        """
        greeting = 'Hello, %s!' % name
        return greeting

    hello() # Hello, World!
    hello('Eric') # Hello, Eric!

Can you tell what this function does? Do you see how to "call" the function? Can you make this function shorter?
