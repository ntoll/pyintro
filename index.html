Naming things


    >>> name = 'Nicholas'
    >>> age = 21
    >>> male = True
    >>> quotation = 'My hovercraft is full of eels'
    >>> name
    'Nicholas'
    >>> age
    21
    >>> male
    True
    >>> quotation

Lists (of simple things)


    >>> shopping_list = ['ham', 'eggs', 'fruit', 'milk']
    >>> shopping_list
    ['ham', 'eggs', 'fruit', 'milk']

Lists remain in order.


    >>> shopping_list[0]
    'ham'
    >>> shopping_list[3]
    'milk'
    >>> shopping_list[4]
    Traceback (most recent call last):
    File "", line 1, in 
    IndexError: list index out of range

Refer to an item by its index. Python counts from 0 (zero).

If you index the wrong thing you get an error.

Lists Can Contain Anything


    >>> random_junk = ['foo', 1.234, False, ['hello', 'world']]
    >>> random_junk
    ['foo', 1.234, False, ['hello', 'world']]
    >>> random_junk[0]
    'foo'
    >>> random_junk[3]
    ['hello', 'world']

Dictionaries of keys & values


    >>> me = {
    ... 'name': 'Nicholas Tollervey',
    ... 'age': 39,
    ... 'languages': ['Python', 'Javascript', 'C#', 'Visual Basic', 'Lisp']
    ... }
    >>> me['name']
    'Nicholas Tollervey'
    >>> me['age']
    39
    >>> me['languages'][0]
    'Python'

A "dict" contains named attributes. Reference each attribute by its key to get the associated value. Create a dict that represents you (don't just use the attributes that I do).
